The Problem

A Series A fintech startup operating across the US, UK, and EU was manually consolidating monthly expense exports from Stripe, QuickBooks, and their internal ledger. Transactions came in USD, GBP, and EUR — with no consistent formatting. The finance team spent 8 hours every month-end just cleaning data before they could produce a single report.

❌ Before — Raw Data Problems
  • Date formats mixed (MM/DD/YYYY, DD-MM-YY, text)
  • Multi-currency amounts with symbols ($, £, €)
  • No FX normalization — all currencies mixed raw
  • Categories inconsistent across 3 source systems
  • Duplicate entries from overlapping exports
  • No budget vs actual comparison or flags
✅ After — Automated Output
  • All dates normalized to ISO format (YYYY-MM-DD)
  • All amounts converted to USD at live FX rates
  • Original currency preserved in separate column
  • Categories mapped to unified taxonomy
  • Duplicates removed automatically
  • Budget vs actual summary with overspend flags

The Python Script
# Multi-Currency Expense Automation Script # Author: Aryan Kasture · Financial Data Automation import pandas as pd import requests from datetime import datetime # ── STEP 1: Load all 3 source files ────────────── files = ['stripe_mar.csv', 'quickbooks_mar.csv', 'ledger_mar.csv'] df = pd.concat([pd.read_csv(f) for f in files], ignore_index=True) # ── STEP 2: Normalize all date formats ─────────── df['date'] = pd.to_datetime(df['date'], infer_datetime_format=True, errors='coerce') df = df.dropna(subset=['date']) # ── STEP 3: Clean amount — strip symbols ───────── df['amount_raw'] = (df['amount'] .astype(str) .str.replace(r'[$£€,\s]', '', regex=True) .astype(float)) # ── STEP 4: Fetch live FX rates, convert to USD ── fx = requests.get('https://api.exchangerate-api.com/v4/latest/USD').json()['rates'] fx_map = {'USD': 1.0, 'GBP': 1/fx['GBP'], 'EUR': 1/fx['EUR']} df['amount_usd'] = df.apply(lambda r: r['amount_raw'] * fx_map[r['currency']], axis=1) # ── STEP 5: Standardize categories ─────────────── cat_map = { 'infra': 'Infrastructure', 'infrastructure': 'Infrastructure', 'mktg': 'Marketing', 'marketing': 'Marketing', 'payroll / hr': 'Payroll', 'rent': 'Operations' } df['category'] = df['category'].str.lower().map(cat_map) # ── STEP 6: Remove duplicates ───────────────────── df = df.drop_duplicates(subset=['date', 'description', 'amount_raw']) # ── STEP 7: Budget vs actual + overspend flags ─── budget_usd = {'Infrastructure': 20000, 'Marketing': 25000, 'Payroll': 120000} summary = df.groupby('category')['amount_usd'].sum().reset_index() summary['budget_usd'] = summary['category'].map(budget_usd) summary['variance'] = summary['amount_usd'] - summary['budget_usd'] summary['flag'] = summary['variance'].apply(lambda x: '⚠ OVER' if x > 0 else '✓ OK') # ── STEP 8: Export clean report ─────────────────── with pd.ExcelWriter('march_report_clean.xlsx') as writer: df.to_excel(writer, sheet_name='Clean Data', index=False) summary.to_excel(writer, sheet_name='Summary (USD)', index=False) print(f"✅ Done. {len(df)} records. All amounts in USD. Report saved.")
